# Удаление сотрудника

### Endpoint

DELETE person/{personId}.

Метод удаления сотрудника по идентификатору.

## Параметры

### Параметры пути

| Параметр пути | Описание                    |
|:--------------|:----------------------------|
| {personId}    | Идентификатор пользователя. |


### Пример запроса

```js

curl -I -X DELETE "https://api.employee/person/123"

```
